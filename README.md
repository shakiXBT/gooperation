# Gooperation

The legendary gobblers are for the people

## Usage

Make sure you have [Foundry](https://github.com/foundry-rs/foundry) installed on your machine

### Setup

```sh
git clone https://github.com/shakiXBT/gooperation.git
cd gooperation
forge install 
```

### Build

```sh
forge build
```

### Run Tests

```sh
forge test
```
